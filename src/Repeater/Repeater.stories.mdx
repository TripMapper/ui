# Repeater

```jsx
import Repeater from 'ui/Repeater';
import Input from './index';

<Repeater
	emptyValue={{ name: '' }}
	name="myRelation"
	fields={[
		(row, name) => <Input name={`${name}.name`} defaultValue={row?.name ?? ''} />,
		(row, name) => <Input name={`${name}.email`} defaultValue={row?.email ?? ''} />,
	]}
/>
```
